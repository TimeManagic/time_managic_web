Time Mangic Web
=============

这是一个Chrome浏览器插件，主旨系通过浏览器收集用户在每天浏览各个网站花费的时间。



Install
-----
*Note: 因使用的脚手架原因，需要配置一下环境，Node的版本可以使用nvm进行切换

> node -v11.15.0  <br>
> npm -v6.7.8

To build, install nodejs first, then run these commands:

```bash
$ npm install & yarn
$ yarn chrome-watch
```
